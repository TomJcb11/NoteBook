String type and method

  • a.replace(x,y) -> Copy : replace every occurence of x by y in a .
  • a[x:y] : return a 'cut' copy of a from the x index to the index just before y .
  • a.extend(b) -> None : add the contains of the b list inside the a list.
  • a.pop(x) -> a : remove x from a (default last item)
  • a.clear() -> None : remove every value from a
  • a.reverse() -> None : reverse the a list
  • a.sort(reverse=False) -> None : sort a by ASC or DESC if reverse TRUE
  • a.copy() -> Copy: return a copy of a with a new id
