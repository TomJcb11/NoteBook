Regular Expression in python

regular expression (regex) are very popular in python to detect,match some pattern

the python re package provide all the method

Methods

Method Param1 Param2 Param3 Description
re.match pattern source N/A is pattern at beginning of source ?
re.search pattern source N/A is pattern in source ?
re.findall pattern source N/A return a list of every match
re.sub pattern repl string Replace every occurence by repl in source
re.split pattern source N/A cut the source on every pattern

Symboles

Symbole Description
. every character except new_line
\d decimal (0-9)
\D non decimal
\w word (character,letter, underscore [a-zA-Z0-9_]
\W non-word
\s space,tab,new_line
\S everything except space

Quantificators

Symbole Signification
* zéro ou plusieurs répétitions
+ une ou plusieurs répétitions
? zéro ou une répétition
{n} exactement n fois
{n,} n fois ou plus
{n,m} entre n et m fois

Anchor and group

Symbole Signification
^ beginning of the string ^abc → match "abc" at start
$ end of the string abc$ → match "abc" at end
\b word boundary \bcat\b → match "cat" alone
\B not a word boundary \Bcat\B → match "cat" inside a word
( … ) capturing group (ab)+ → capture "ab" repeated
(?: … ) non-capturing group (?:ab)+ → group without capturing
pipe OR / alternatives

Importance of raw string

In python we use '\' in string like : - \t => tabulation - \n => new_line we don't want this in our pattern matching expression

#witout raw string
pattern = "\\b\w+@\\w+\\.\\w+\\b"  # il faut doubler les backslashes
#with raw string
pattern = r'\b\w+@\w+\.\w+\b'  # plus lisible et simple

Regex use case

there are many use case for regular expression.

Counting Occurrences of a pattern in a text

def count_character(char:str,string:str):
   return len(re.findall(pattern=char,string=string))
print(count_character("b","blelblelelelelele"))
output => 5

Checking type inside a string

def is_text_only(string:str):
    return not bool(re.search(r'[^a-zA-Z]',string=string))
text1="HelloWorld" #=> True
text2="HelloWorld2" #=> False

Validation of e-mails

def validate_email(string):
    pattern=r'^[\w]+@[\w.]+\.[a-zA-Z]{2,3}$' #"username@domain.com":True
    strict_pattern=r'^[a-zA-Z0-9]+\.?[a-zA-Z0-9]+@[a-zA-Z]+(\.[a-zA-Z]+)*\.[a-zA-Z]{2,10}$' # a more strict version of the pattern
    return bool(re.match(pattern=strict_pattern,string=string))
